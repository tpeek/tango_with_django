About This Project

This project was a coding challenge given to me by the Code Fellows team. I was to create a Django app that allows a user to create, edit, update, and delete other users. Users are required to have a first name, optional last name and valid email address. I added in a username and password for login functionality, along with an optional website and picture just for fun.
